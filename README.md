# Factorial Calculator

This repository contains a simple Python script to calculate the factorial of a given non-negative integer.

## Files

- `factorial.py`: The main Python script containing the factorial logic.
- `README.md`: This file.

## How to Run

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/raushan291/test-repo.git
    cd test-repo
    ```

2.  **Run the Python script:**
    ```bash
    python factorial.py
    ```

    The script will prompt you to enter a non-negative integer, and then it will display its factorial.

## Example

```
Enter a non-negative integer: 5
The factorial of 5 is 120
```
